1. Tujuan

Setelah mempelajari modul ini, peserta didik diharapkan dapat:
• Memahami dasar pemrograman web menggunakan HTML (HyperText Markup Language).
• Mengenal berbagai tag dan elemen dasar HTML.
• Menyusun halaman web sederhana menggunakan HTML.
• Menyelesaikan beberapa latihan pemrograman dasar HTML secara mandiri.

2. Dasar Teori

2.1 Pengertian HTML

HTML (Hyper Text Markup Language) Merupaka bahasa markup untuk menyusun halaman web yang dapat dibaca oleh browser.

Browser seperti Google Chrome atau Firefox akan menerjemahkan kode HTML menjadi tampilan visual halaman web

Beberapa Element Dasar HTML yang sering digunakan :

Tag HTML Fungsi/Deskripsi
<h1> – <h6> Menentukan ukuran judul (heading)
<p> Menulis paragraf
<a> Membuat tautan (link)
<div> Membuat blok atau bagian tertentu dalam halaman
<img> Menampilkan gambar
<br> Ganti baris
<hr> Menambahkan garis horizontal
<b> Menebalkan teks
<i> Memiringkan teks
